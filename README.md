# Introdução do Projeto 
No dia de hoje iremos aprender como se usar o VS code e o GitHub.
## Processo de criação 
- Crie uma pasta em seu gerenciar dor de arquivos. 
- Abra o VS code e abra sua pasta no VS code.
- para abrir basta clicar no botão file/pasta, ir em open folder e achar a sua pasta e abri ela.
## Finalização 
- abra seu git hub ou crie sua conta no mesmo.
- crie um repositorio/commit.
- apos cria o commit ache uma mensagem em azul escrito uploading an existing file
- depois coloque sua pasta, que você criou la no inicio, adicionando ela como um projeto no github.
